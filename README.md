### 🚀 Projetos em destaque

#### 🥊 [UFC Fight Predictor](https://github.com/henriquepicchioni/ufc_machine-learning)(Em desenvolvimento...)
Sistema de machine learning que prevê o resultado de lutas do UFC a partir das
estatísticas dos lutadores. Cobre o fluxo completo — limpeza de dados, análise
exploratória, construção de features, treino e avaliação do modelo, e uma interface web.

`Python` · `pandas` · `scikit-learn` · `Streamlit`

#### 🎴 [Slay the Spire-inspired Game](https://github.com/henriquepicchioni/LikeSlayTheSpire)
Jogo *roguelike deck-builder* em turnos, inspirado no Slay the Spire, desenvolvido em C
com Allegro 5 como trabalho final da disciplina de Programação (PDS1) na UFMG. O jogador
usa cartas de ataque, defesa e especiais para vencer inimigos enquanto sobe uma torre —
o objetivo é sobreviver a 10 batalhas seguidas; perder uma única luta encerra a run.

> 🎓 Primeiro projeto desenvolvido na faculdade — marca o início da minha jornada na Ciência da Computação.

`C` · `Allegro 5`
